How is this template modular?

How are dropdown menus accessible?

Menus support keyboard controls, proper aria-expanded states and Escape-to-close behavior.

Can we remove JavaScript entirely?

Yes. The page still renders without JS; dropdowns simply remain collapsed and can be expanded in fallback builds.
